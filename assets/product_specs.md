# Product Specifications - E-Shop Checkout

## Discount Codes
- The discount code SAVE15 applies a 15% discount.
- The discount code WELCOME10 applies a 10% discount.
- Invalid or expired discount codes must display an error message: “Invalid Discount Code”.

## Shipping
- Standard Shipping is FREE.
- Express Shipping costs $10.

## Cart Rules
- Users can update item quantities.
- Total price must update in real-time.
- If quantity is set to 0, item is removed from cart.

## Payment
- Supported payment methods: Credit Card, PayPal.
- “Pay Now” button must only activate if all required fields are valid.
